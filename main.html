Recent Projects

VIMusic

This project involved setting up an informational website with ecommerce functionality, enabling the user to purchase "learn music" courses. This was achieved using WooCommerce and WP-CourseWare. The main focus was to enable the site to be accessible to the widest audience possible from the standpoint of ability. I worked primarily on visual design, keyboard/swipe accessibility & disability testing.
